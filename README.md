## 📦 What is Makine-Launcher?

Makine-Launcher is a Windows app for Turkish game localization projects. It helps you find, download, and open supported translation tools from one place. The app uses Qt6, QML, and C++23, but you do not need to know any of that to use it.

You can use it to:
- browse Turkish translation projects
- check available launcher updates
- start supported tools from a single screen
- keep game translation files in one place
- access community-made translation content with less effort

## 🧭 Troubleshooting

### The file does not open
- Check that the download finished.
- Right-click the file and choose Open.
- If Windows blocks it, choose Run anyway if you trust the file source.

### The launcher does not start
- Restart your PC.
- Try opening the app as an administrator.
- Make sure your Windows version is up to date.

### A project does not appear
- Check your internet connection.
- Refresh the launcher.
- Look for a newer release on the release page.

### The game files seem wrong
- Remove the files you just added.
- Reinstall the translation pack from the launcher.
- Check that you selected the correct game folder.
